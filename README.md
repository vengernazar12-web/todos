# Todo App

Простий Todo-додаток на Vanilla JavaScript.

## Функціонал
- Додавання todo
- Видалення з анімацією
- Редагування
- Позначення виконаних
- Пошук по todo
- Сортування (виконані вниз)
- Збереження в localStorage

## Технології
- HTML
- CSS (flex, clamp, :has)
- JavaScript (без бібліотек)

## Запуск
Відкрити `index.html` у браузері
